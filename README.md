# Coxeter Groups in Lean

We formalize in Lean some results on Coxeter groups, following the textbook *Combinatorics of Coxeter Groups* by A. Björner and F. Brenti. Some results currently available in this repository:

* Construction of the permutation repersentation
* Strong exchange
* Deletion property
* Definition of the Bruhat order
* Subword property
* Lifting property
* The longest element of a finite Coxeter group
